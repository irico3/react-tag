# メッセージをタグ毎に整理するreact
こちらのリポジトリはreact学習用です。


## 開発環境

- node v12.13.1
- npm v6.13.4

### install

```
$ npm ci
```

### usage

```
$ npm run start
```

### build

```
$ npm run build
```

### test

```
$npm t
```

### 構成

react / redux-tool-kit を使用

react v16.13.1,
reduxjs/toolkit v1.3.6
